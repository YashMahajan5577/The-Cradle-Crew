
A. Start Server
    1. create new terminal
    2. run command `cd server`
    3. run command `npm install`
    4. run command `node index.js`
    5. server will start and data base connect itself

B. Start App
    1. create new terminal
    2. cd client
    3. run command `npm install`
    4. run command `npm start`
    5. application will start in default browser
